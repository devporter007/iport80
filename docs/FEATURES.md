# Features
Currently the following features are supported as of now in the emulator:
* Instruction Set: 
    * ACI
    * ADC R/M
    * ADD R/M
    * ADI
    * LDA
    * LDAX
    * LHLD
    * LXI RP, D16
    * MOV R,R | R,M | M,R
    * MVI R/M
    * SBB
    * SBI
    * SHLD
    * STA
    * STAX
    * SUB R/M
    * SUI
    * XCHG
  
* Input:
  * Currently utilises input.txt for loading the memory of emulator with both executable code and data if needed.
